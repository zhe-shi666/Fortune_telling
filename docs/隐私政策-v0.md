# 隐私政策 v0

本草稿用于当前 MVP 阶段，帮助后续接入正式隐私政策页面与 App Store Connect 隐私信息。

## 1. 适用范围

本应用当前为命理娱乐参考类产品，提供今日参考、八字参考、取名参考、合婚参考、命主档案与未来内购入口等功能。

## 2. 当前会处理的信息

- 命主档案信息：
  - 出生日期
  - 出生时辰
  - 性别
  - 历法类型
- 取名输入信息：
  - 可选姓氏
  - 性别
  - 出生日期
  - 出生时辰
- 本地偏好与使用信息：
  - 收藏的名字
  - 充值页中的方案与支付方式选择状态

## 3. 当前数据边界

- 上述信息当前默认保存在设备本地，用于页面展示、输入回填和功能连续性。
- 当前版本不要求注册登录，不建立账号体系。
- 当前版本不提供多端同步，不将收藏名字或命主档案与远端账号绑定。
- 当前版本的购买按钮点击后仅展示购买说明文案，不会触发真实支付。

## 4. 当前不做的事情

- 不接入第三方广告 SDK。
- 不出售或共享用户个人信息。
- 不基于命主档案做定向广告投放。
- 不在当前版本中接入真实支付订单、退款或财务账本能力。

## 5. 内容说明

- 今日、八字、取名、合婚结果当前统一作为娱乐参考或辅助参考展示。
- 本应用中的内容不构成专业建议、医疗建议、法律建议或财务承诺。

## 6. 后续正式版需要补齐

- 正式隐私政策 URL
- 联系方式与数据删除渠道
- App Store Connect 中的 App Privacy 申报
- 若未来接入真实内购、账号或后端服务，对数据流转范围的补充说明
